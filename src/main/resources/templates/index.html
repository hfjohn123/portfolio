Kinship Classification Ablation Study

This project is sponsored by Indago LLC.
The data set we used for this project is FIW.
In this project, we applied transfer learning on the dataset with paired images of portraits. We analyze how the imbalanced dataset would affect the result. The resulting models can accurately classify the kinship between two blood-related persons.
You can click on the image for report. And there is a link toward the dataset and notebook.

Rurl—— Short Url Generator

This website is a short url generator that transforms your long url into a short one so you can share it with others more easily.
And as an example, this Demo is collecting UA (user agent) info from the request. This is an example for how could a website like this benefits the business.
MVC and HTTP Server by SpringBoot. Spring Data as ORM and MongoDB for data Storage. Design and function based on Restful API.
Click the image to get to the Demo. you can achieve the source code through the navbar of the demo site.

Rurl—— 短链接生成器

这个网站是个短链接生成器，可以把长链接转换成短链接，这样用户可以更简单的把它分享给其他人。
这里是一个例子，这个Demo会从Request中收集UA (user agent)信息，这是这个网站的运行者获益的一种方式。
SpringBoot的MVC和HTTP服务器。Spring Data作为ORM，MongoDB用来存储数据。基于Restful API设计和工作。
点击图片跳转到Demo。如果你想要查看源代码。你可以从Demo的Navbar里边的超链接找到对应的GitHub。
